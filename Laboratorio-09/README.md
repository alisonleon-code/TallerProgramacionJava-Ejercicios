# Laboratorio 09 – Arreglos de una Dimensión(Vectores)

## Descripción
Este laboratorio tiene como objetivo aplicar el uso de arreglos de una dimensión (vectores) en Java, permitiendo almacenar, manipular y procesar conjuntos de datos del mismo tipo.

Se desarrollan ejercicios donde se trabaja con vectores para realizar operaciones como filtrado, búsqueda, transformación de datos y estadísticas básicas, utilizando estructuras repetitivas y condicionales.

## Estructura del laboratorio
El laboratorio se encuentra organizado en las siguientes carpetas:

**codigo**  
Contiene los programas desarrollados en Java, donde se implementan los ejercicios utilizando arreglos (vectores).

**enunciado**  
Contiene el documento PDF del laboratorio con los problemas propuestos.

**proyecto**  
Contiene el proyecto completo comprimido (.zip) listo para ejecutar en NetBeans u otro IDE.

## Contenido del laboratorio

- Declaración y uso de arreglos (vectores)
- Recorrido de vectores con estructuras for
- Validación de datos en arreglos
- Uso de vectores auxiliares
- Filtrado de elementos según condición
- Transformación y modificación de vectores
- Aplicación de lógica condicional en estructuras de datos
- Resolución de problemas tipo estadística y análisis de datos

## Ejercicios desarrollados

- Ejemplo 1: Inicialización de vectores
- Ejemplo 2: Ingreso de datos en vectores
- Ejercicio 1: Filtrado de edades mayores o iguales a 18
- Ejercicio 2: Reemplazo del menor elemento con suma posterior
- Ejercicio 7: Conteo y porcentaje de vocales en un vector
- Ejercicio 9: Reemplazo basado en múltiplos de 7
- Ejercicio 11: Unión de vectores con números pares
- Ejercicio 12: Operaciones entre dos vectores (suma y resta)

## Tecnología utilizada
- Lenguaje de programación: Java  
- IDE: Apache NetBeans
