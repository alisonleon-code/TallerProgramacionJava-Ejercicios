# Laboratorio 11 – Arreglos de dos Dimensiones (Matrices)

## Descripción  
Este laboratorio tiene como objetivo aplicar el uso de arreglos de dos dimensiones (matrices) en Java, permitiendo almacenar, manipular y procesar datos organizados en filas y columnas.

Se desarrollan ejercicios donde se trabaja con matrices para realizar operaciones como recorridos, sumas, intercambios, búsquedas, filtrado de datos y cálculos estadísticos, utilizando estructuras repetitivas anidadas y condicionales.

---

## Estructura del laboratorio  

El laboratorio se encuentra organizado en las siguientes carpetas:

### codigo  
Contiene los programas desarrollados en Java, donde se implementan los ejercicios utilizando matrices.

### enunciado  
Contiene el documento PDF del laboratorio con los problemas propuestos.

### proyecto  
Contiene el proyecto completo comprimido (.zip) listo para ejecutar en NetBeans u otro IDE.

---

## Contenido del laboratorio  

- Declaración y uso de arreglos bidimensionales (matrices)  
- Recorrido de matrices con for anidados  
- Validación de datos en matrices  
- Uso de vectores auxiliares  
- Filtrado de elementos según condición  
- Transformación y modificación de matrices  
- Intercambio de filas, columnas y diagonales  
- Cálculo de sumas, promedios y estadísticas  
- Resolución de problemas con lógica matricial
