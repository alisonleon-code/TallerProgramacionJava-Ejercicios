# Laboratorio 07 – Estructura Repetitiva DO WHILE

## Descripción
Este laboratorio tiene como objetivo aplicar la estructura repetitiva `do while` en Java para resolver distintos problemas mediante ciclos controlados por condición.

A través de los ejercicios se desarrollan procesos de repetición para el ingreso de datos, cálculos, validaciones y generación de reportes, ejecutando al menos una vez el bloque de instrucciones antes de evaluar la condición.

Además, se refuerza el uso de:
- Contadores
- Acumuladores
- Banderas de control
- Cálculo de promedios
- Búsqueda de máximos y mínimos
- Uso de ciclos anidados
