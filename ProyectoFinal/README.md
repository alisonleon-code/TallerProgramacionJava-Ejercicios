Proyecto: Gestión de Venta de Pasajes Aéreos

Descripción del Proyecto:
Este es un proyecto de desarrollo de una aplicación de escritorio en Java, diseñada para automatizar el proceso de venta de boletos aéreos para la aerolínea JetSMART Perú. El sistema calcula automáticamente el precio final de los boletos, considerando destino, tipo de clase, equipaje, y descuentos por cantidad de boletos adquiridos.

Objetivo del Proyecto

El objetivo de este proyecto es digitalizar y automatizar el proceso de compra de boletos aéreos en JetSMART Perú. A través de una aplicación basada en Java y utilizando programación orientada a objetos, el sistema:

Registra los datos de los pasajeros.
Calcula los precios según el destino, tipo de boleto y equipaje.
Aplica descuentos según la cantidad de boletos adquiridos.
Genera boletas detalladas y resúmenes finales de la venta.
Cómo Funciona la Aplicación
Ingreso de Datos: El usuario introduce el número de boletos a emitir, junto con los datos de cada pasajero (nombre, destino, tipo de boleto y equipaje).
Cálculo de Precios: Basado en el destino seleccionado, el tipo de boleto (básico, turista, primera clase), y el equipaje (con recargos si excede los 23kg), el sistema calcula el precio total de cada boleto.
Generación de Boleta: Al finalizar, se genera una boleta detallada por cada pasajero, mostrando el precio base, los aumentos por tipo de clase, los recargos por equipaje y el descuento aplicado.
Resumen General: Al final del proceso, se muestra un resumen general con el precio total sin descuento, los descuentos aplicados y el monto final a pagar.
Limitaciones Actuales
Sin interfaz gráfica (GUI): La aplicación es completamente de consola.
Sin integración con pagos en línea: No se maneja la parte de pagos ni integraciones web.
Limitación de funcionalidades: No permite gestionar cambios, anulaciones ni múltiples maletas.
