# Laboratorio 14 – Modificadores de Acceso

## Descripción
Este laboratorio tiene como objetivo aplicar el uso de modificadores de acceso en Java mediante encapsulamiento, utilizando atributos privados, métodos getters y setters, y métodos para procesar información desde clases externas.

Se desarrollan ejercicios orientados al uso de clases, objetos, acceso controlado a atributos y resolución de problemas aplicando Programación Orientada a Objetos.

## Estructura del laboratorio

El laboratorio se encuentra organizado en las siguientes carpetas:

### codigo
Contiene los programas desarrollados en Java usando clases, atributos encapsulados, getters, setters y métodos de cálculo.

### enunciado
Contiene el documento PDF del laboratorio con los problemas propuestos.

### proyecto
Contiene el proyecto completo comprimido (.zip) listo para ejecutar en NetBeans u otro IDE.

## Contenido del laboratorio
- Modificadores de acceso (private)
- Encapsulamiento de atributos
- Métodos getters y setters
- Uso de la palabra reservada `this`
- Métodos de acceso y modificación
- Programación orientada a objetos aplicada
- Validaciones y cálculos mediante métodos
- Aplicación de estructuras condicionales y repetitivas

## Ejercicios desarrollados

### Ejemplo
- Ejemplo 1: Cálculo de suma usando getters y setters

### Ejercicios propuestos
- Ejercicio 1: Conteo de múltiplos de 3, 4 y 7
- Ejercicio 2: Compra de pasajes con descuentos por cantidad
- Ejercicio 3: Estadísticas de empleados (salarios y conteo por sexo)
- Ejercicio 4: Sueldo según contrato y estado civil
- Ejercicio 5: Pago de obrero por producción y bonificación
- Ejercicio 6: Compra de pantalones con descuentos por cantidad
- Ejercicio 7: Promedio de pesos por edades
- Ejercicio 8: Pago mensual de socios por categoría y descuento

## Tecnología utilizada
