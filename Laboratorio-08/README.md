# Laboratorio 08 – Estructura Repetitiva FOR

## Descripción
Este laboratorio tiene como objetivo aplicar la estructura repetitiva `for` en Java para resolver distintos problemas mediante ciclos controlados por contador.

A través de los ejercicios se desarrollan procesos de repetición para ingreso de datos, cálculos, validaciones, series numéricas, uso de ciclos anidados y generación de reportes.

Además, se refuerza el uso de:

- Contadores  
- Acumuladores  
- Validaciones  
- Cálculo de porcentajes y promedios  
- Método del supuesto falso (máximos y mínimos)  
- Uso de ciclos `for` simples y anidados  

---

## Estructura del laboratorio

El laboratorio se encuentra organizado en las siguientes carpetas:

### codigo
Contiene los programas desarrollados en Java (`Ejercicio01` hasta `Ejercicio16`) aplicando estructuras repetitivas `for`.

### enunciado
Contiene el documento PDF del laboratorio con los problemas planteados.

### proyecto
Contiene el proyecto completo comprimido (`.zip`) para su ejecución en el IDE.

## Contenido del laboratorio

- Uso de la estructura repetitiva `for`
- Ciclos controlados por contador
- Validación de datos dentro de ciclos
- Uso de acumuladores y contadores
- Aplicación de máximos y mínimos
- Cálculo de porcentajes y promedios
- Uso de ciclos anidados (`for` dentro de `for`)
- Desarrollo de series numéricas
- Resolución de problemas tipo casos reales:
  - Cursos electivos
  - Descuentos por edades
  - Sueldos por horas
  - Registro de empleados
  - Series matemáticas
  - Fábrica de cajas
  - Supermercado
  - Encuestas
  - Aulas y alumnos
  - Venta de autos
  - Competencia de atletas
  - Números primos
  - Transporte de carga
  - Divisores de un número
