# Laboratorio 13 – Parámetros y Argumentos

## Descripción
Este laboratorio tiene como objetivo aplicar el uso de **parámetros, argumentos, clases y métodos en Java**, desarrollando programas estructurados mediante programación orientada a objetos.

Se trabajó con métodos de tipo **void** y métodos con **return**, aplicando cálculos, validaciones y modularización mediante clases auxiliares y clases principales.

---

## Estructura del laboratorio

El proyecto se encuentra organizado en las siguientes carpetas:

### codigo
Contiene los programas desarrollados en Java para los ejemplos y ejercicios propuestos.

### enunciado
Contiene el documento PDF del laboratorio con la guía y problemas planteados.

### proyecto
Contiene el proyecto completo comprimido (.zip) listo para ejecutar en NetBeans.

---

## Contenido del laboratorio

- Uso de clases y objetos
- Parámetros y argumentos
- Métodos tipo void
- Métodos con retorno (return)
- Variables globales y locales
- Modularización mediante métodos
- Uso de condicionales y estructuras switch
- Desarrollo de programas orientados a objetos
