# Laboratorio 03 – Estructura Condicional IF-ELSE

## Descripción

Este laboratorio tiene como objetivo practicar el uso de la estructura condicional **if - else** en Java. Se desarrollan ejemplos y ejercicios donde se aplican condiciones para tomar decisiones y obtener resultados según distintos casos.

---

## Estructura del laboratorio

El laboratorio se encuentra organizado en las siguientes carpetas:

### codigo

Contiene los programas desarrollados en Java, incluyendo los ejercicios propuestos resueltos utilizando estructuras condicionales `if - else`.

### enunciado

Contiene el documento PDF del laboratorio, donde se presentan los problemas y ejercicios a resolver.

### proyecto

Contiene el proyecto completo comprimido (.zip) para poder abrirlo directamente en el IDE.

---

## Contenido del laboratorio

* Uso de la estructura condicional `if - else`
* Aplicación de condiciones simples y compuestas
* Resolución de problemas mediante lógica condicional
* Uso de variables y operadores lógicos

---

## Tecnología utilizada

* Lenguaje de programación: Java
* IDE: Apache NetBeans
