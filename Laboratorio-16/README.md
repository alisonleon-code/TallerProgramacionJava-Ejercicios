# Laboratorio 16 – Sobrecarga de Constructores y Métodos

## Descripción
Este laboratorio tiene como objetivo aplicar los conceptos de sobrecarga de constructores y sobrecarga de métodos en Java, permitiendo reutilizar nombres de métodos y constructores variando la cantidad o tipo de parámetros.

Se desarrollan ejercicios donde se emplean métodos sobrecargados para resolver distintos problemas de cálculos, tarifas, promedios, pagos y operaciones diversas.

## Estructura del laboratorio

El laboratorio se encuentra organizado en las siguientes carpetas:

### codigo
Contiene los programas desarrollados en Java utilizando sobrecarga de métodos y constructores.

### enunciado
Contiene el documento PDF del laboratorio con los problemas propuestos.

### proyecto
Contiene el proyecto completo comprimido (.zip) listo para ejecutar en NetBeans u otro IDE.

## Contenido del laboratorio

- Sobrecarga de constructores  
- Sobrecarga de métodos  
- Uso de parámetros y argumentos  
- Aplicación de `this` en constructores  
- Métodos con retorno y tipo void  
- Reutilización de métodos con distintas firmas  
- Resolución de problemas mediante programación orientada a objetos

## Ejercicios desarrollados

### Ejemplos
- Ejemplo 1: Área de un rectángulo con sobrecarga de constructores  
- Ejemplo 2: Área de cuadrado y rectángulo con sobrecarga de métodos

### Ejercicios
- Ejercicio 1: Datos del obrero y cálculo de sueldo final  
- Ejercicio 2: Boleta de empleado con aumento salarial  
- Ejercicio 3: Promedios de prácticas y exámenes  
- Ejercicio 4: Cálculo de tarifa telefónica con descuentos por zona  
- Ejercicio 5: Pago final de productos según modelo y talla

## Tecnología utilizada

- Lenguaje de programación: Java  
- IDE: Apache NetBeans
