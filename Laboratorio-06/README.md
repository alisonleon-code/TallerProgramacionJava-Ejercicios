# Laboratorio 06 – Estructura Repetitiva WHILE

## Descripción

Este laboratorio tiene como objetivo aplicar la estructura repetitiva **while** en Java para resolver distintos problemas mediante el uso de ciclos. Se desarrollan ejercicios donde se repiten procesos de ingreso de datos, cálculos y validaciones hasta que el usuario decide finalizar.

Además, se refuerza el uso de acumuladores, contadores y el **método del supuesto falso** para determinar valores máximos y mínimos dentro de un conjunto de datos.

---

El laboratorio se encuentra organizado en las siguientes carpetas:

### enunciado

### proyecto

---

## Contenido del laboratorio

* Uso de la estructura repetitiva `while`
* Ciclos controlados por condición (`while(preg == 1)`)
* Uso de acumuladores y contadores
* Validación de datos dentro de ciclos
* Uso de ciclos anidados (`while` dentro de `while`)
* Aplicación del método del supuesto falso (máximos y mínimos)
* Resolución de problemas tipo casos reales (ventas, encuestas, registros)
