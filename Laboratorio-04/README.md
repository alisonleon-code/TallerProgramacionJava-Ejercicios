# Laboratorio 04 – Estructura Condicional SWITCH

## Descripción

Este laboratorio tiene como objetivo aplicar la estructura condicional **switch** en Java para resolver distintos problemas mediante selección múltiple. Se desarrollan ejercicios donde se evalúan diferentes opciones y se ejecutan acciones específicas según el caso ingresado.

---

## Estructura del laboratorio

El laboratorio se encuentra organizado en las siguientes carpetas:

### codigo

Contiene los programas desarrollados en Java, incluyendo los ejercicios resueltos utilizando la estructura `switch` combinada con `if - else`.

### enunciado

Contiene el documento PDF del laboratorio, donde se presentan los problemas planteados.

### proyecto

Contiene el proyecto completo comprimido (.zip) para su ejecución en el IDE.

---

## Contenido del laboratorio

* Uso de la estructura condicional `switch`
* Combinación de `switch` con `if - else`
* Evaluación de múltiples opciones
* Resolución de problemas aplicando lógica de programación

---

## Tecnología utilizada

* Lenguaje de programación: Java
* IDE: Apache NetBeans
